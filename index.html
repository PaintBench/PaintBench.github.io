PaintBench

Tasks trivial in any raster editor, still out of reach for frontier multimodal models.

A dynamically scalable benchmark spanning 20 primitive visual editing operations. Procedural generation enables ∞ problem variety, preventing saturation. Deterministically evaluated with pure pixel math — no biased model judges.

The highest-performing model we evaluated (Nano Banana 2) scores only 23.5%.

Paper GitHub Dataset Results

Primitive Operations

20 atomic task types under 4 categories: geometric transformation, structural manipulation, color change, and symbolic reasoning.

Deterministically Evaluated

Like many real-world editing tasks, problems have a single correct output. No model judges, no subjectivity — just math.

Dynamically Scalable

Generated from random seeds. New problem instances are always available, so performance reflects genuine capability.

TinyGrafixBench

Can failures on geometric primitives predict performance on real-world charts? We applied PaintBench's framework to data visualization editing.

600 problems across 5 chart types and 4 editing operations each, procedurally generated and deterministically evaluated.

Model rankings strongly correlate between PaintBench and TinyGrafixBench, validating our primitive benchmark as a proxy for harder real-world tasks.
