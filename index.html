PaintBench

Tasks trivial in any raster editor, still out of reach for frontier AI.

A dynamically scalable benchmark spanning 20 primitive visual editing operations found across real-world workflows. Problems are procedurally generated, preventing saturation. Evaluation is pure pixel math, with no model judges.

The best image-editing model scores only 23.5%.

Geometric Structural Color Symbolic

Primitive Operations

20 atomic task types across 4 categories: geometric transformation, structural manipulation, color change, and symbolic reasoning.

Deterministically Evaluated

Every problem has exactly one correct output. Pure pixel math, like HumanEval for code but for images. No model judges, no subjectivity.

Dynamically Scalable

Generated from random seeds with tunable parameters. Fresh unseen problems can be generated at any time, preventing benchmark saturation.

Model Performance

mIoU (%): mean Intersection-over-Union integrated over 11 CIE ΔE₇₆ thresholds. Bars are out of 100% to show how much headroom remains. Click a row to expand per-category scores.

Companion Benchmark

We also built
TinyGrafixBench

Can failures on geometric primitives predict performance on real-world charts? We applied PaintBench's framework to data visualization editing to find out.

TinyGrafixBench is a 600-problem benchmark across 5 chart types, 4 editing operations each, procedurally generated and deterministically evaluated.

Model rankings strongly correlate between PaintBench and TinyGrafixBench, validating our primitive benchmark as a proxy for harder real-world tasks.
